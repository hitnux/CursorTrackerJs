Examples

Options

Title
Type
Description
id
String
The id of the tracker element

Default:"cursor-tracker"
container
String
Target container classes

Default:".ct-container"
selector
String
Target content class

Default:".ct-content"
top
Number
Top space of the Tracker

Default:5
left
Number
Left space of the Tracker

Default:0
active
String
The class that the active container will receive

Default:"active"
breakpoint
String
Minimum device width

Default:"1024px"
innerHtml
String
Html of tracker element
classes
String
The classes of the tracker element
    const tracker = new CursorTracker({
        id: 'tracker',
        container: '.container .box',
        selector: '.content',
        top: 25,
        left: 15,
        active: 'active',
        breakpoint: '1024px',
        innerHtml: 'Cursor is here',
        classes: 'cursor-tracker'
    });

Functions

Title
Description
onTheElement
Tracking a custom item
    const pause = document.getElementById('pause');
    tracker.onTheElement({ target: pause, classes: 'hidden' });

Events

Title
Description
activated
Event to run when container is active
deactivated
Event to run when the container is deactivated
moving
Event to run while tracker is moving
    document.querySelector('.container').addEventListener('activated', (e) => { console.log(e) })
